# Lijiajie (李嘉杰) — Personal Homepage

A bilingual personal homepage hosted on GitHub Pages.

## Contents

- `index.html` — English version
- `images/lijiajie.jpg` — Profile photo

## Usage

Open `index.html` in your browser for the Chinese version, or `index_en.html` for the English version. Click the language switch button in the top-right corner to toggle between them.

## Tech

Pure HTML + CSS + JavaScript. No build tools or frameworks required.
